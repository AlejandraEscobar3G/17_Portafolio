Laboratoria

Nuestro Portafolio

1_CONOCIENDO HTML

En este ejercicio se practaron las etiquetas básicas de HTML y en días posteriores se incluyeron fuentes al HTML así como también imágenes con link a otros sitios web, se utilizó la propiedad target, asi que las paginas se abren en una nueva pestaña.

2_ViendoCajas

Podemos observar que estamos utilizando etiquetas tipo bloque, de esta manera las coders empezarán a diferenciar entre block e inline.

3_LenguajesDeLaWeb

En esta sección se vio la importancia de tener un archivo css externo en vez de poner estilos adentro de la etiqueta head o de manera inline en el html.

4_Cajas

En el ejercicio cajas empezamos a utilizar padding para mejorar la distribución y el diseño de nuestro contenido.

5_OchoCajas

Es importante saber que las cajas tipo bloque, pueden permitir que más elementos se coloquen a su lado, por ello, se hizo uso de la propiedad display que nos permite acomodar uno o más elementos a lado de otro de tipo bloque.

6_Float

Y no sólo podemos colocar elementos a lado de otros, sino que también los podemos flotar hacia el lado que deseemos (todos a la izquierda, o bien, todos a la derecha)

7_Positioning

¿Qué ocurre si queremos colocar un elemento en la esquina inferior derecha? Para no hacer las cosas tan tediosas, HTML nos ofrece la propiedad de position con sus 3 valores, ahora las coders pueden usar esta propiedad de acuerdo a lo que necesitan.

8_NuestrasCoders

Aqui únicamente se combinaron las propiedades de display y position, para aprender que ambas propiedades trabajando en conjunto, hacen muchas cosas.

9_LaCancha

Aqui tenemos una pequeña práctica en la que tuvimos que evaluar lo que necesitabamos para poder resolver el problema de la cancha.

10_MiPrimerLayOut

Position, Display...y la Grid System. Este es el primer ejercicio en el que nosotras incorporamos un nuevo término y una nueva herramienta.

11_GrydSystem

Ejercicio en el que tenemos más terminos nuevos y trabajamos con la grid system para tener un mejor entendimiento de esta gran herramienta.

12_GrydSystem-2

Al inicio, la herramienta de la grid system es un poco difícil de entender porque no logramos visualizarla, pero en este ejercicio se puede apreciar claramente como es que funciona la grid (renglones y columnas).

13_GrydEjercicios

Además hay muchas aplicaciones para la grid, podríamos incluso hacer con ella la página de algún periódico, hay muchas maneras de distribuir el contenido y este es sólo uno de muchos ejemplos.

14_Cinema

Es importante planear las distribuciones antes de comenzar a codear tu nueva página, en este ejercicio las coders aprendimos esa gran importancia y lo pusimos en práctica.

15_Chicama

Una vez más... planear es importante. Segundo ejercicio en el que trabajamos planeando antes de comenzar a codear.

16_MiPerfil

Las coders ya tenemos un mejor entendimiento del uso de la grid system, hemos estado haciendo pequeños proyectos de maquetado, en este punto ya es más sencillo observar la distribucíón de la página e irla dividiendo para que se lea de manera más sencilla y se le puedan dar mejores estilos.
